# Scraping_project

Projet Scrapy pour scrapper des produits sur boulanger.com.

## Structure
- spiders/ : Contient les spiders Scrapy
- items.py : Définit les modèles de données
- pipelines.py : Définit les pipelines de traitement
- settings.py : Paramètres du projet

## Lancement

```bash
scrapy crawl boulanger_spider
```
